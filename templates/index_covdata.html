Covdata Documentation

This covidindia Module has the power to bring all the data related to Covid-19 outbreak in India in a tabular form. This module will bring the whole set of raw data in a date-wise and processed format from the COVID-19 Indian Dataset maintained by University of Kalyani, which will be easier to deal with. Once installed, we can get an up-to-date information about the Covid-19 affected people, till the day the data is available at the source. We can visualize the data of our choice for any given range of dates. It is capable of giving information about the total number of confirmed, recovered and deceased patients, between any two dates, of any city, district or related to any particular gender (male/female). We can also visualize the whole up-to-date data without using any aforesaid filters using our module.

Usage

It gathers information from our own maintained database. Database data are collected from various websites, news channels and covid19india.org crowdsourcing initiative. From the collected data this module will help user to represent in user friendly format with all types of filters in one place. No need to search any website. Just use this module for India based Covid-19 projects.

Installation

For windows: pip install covdata

For mac:pip3 install covdata

Initializing covdata

At first initialize the covidindia module to collect all data from the source.It is very important as all other classes are dependent on this class instance.To do that write the following code

from covdata.covidindia import initializer

init=initializer(silent=False)   #It will make a dashboard type display in console. Use silent to be true to print nothing in console

collectedData=init.show_data(of='Confirmed',daily=False)   #This will show the collected Confirmed data

collectedData=init.show_data(of='Confirmed',daily=True)   #This will show the collected Confirmed data as daily count basis

*Here of argument can take 3 values Confirmed,Recovered,Death to see all respective COVID-19-Datasets

Demography Data

In that class user can see the count of affected patients from different states, districts and cities along with affected date and gender of the patients.To do this write the following

from covdata.covidindia import initializer,Demographic_overview

init=initializer(silent=False)

data=Demographic_overview(init)   #create instance object of Demographic_overview class

wb_data=data.demography(place='wb',date='1/06/2020')   #gives the data of West Bengal of 1/06/2020.State code or state name can also be used

nagpur_data=data.demography(place='Nagpur',date='all')   #gives the data of nagpur for all dates

all_data=data.demography()  #gives all the data for all states and all dates

between_data=data.demography(place='mh',date='5/5/2020-10/5/2020')   #gives the data of Maharashtra between 5/5/2020 and 10/5/2020

*place argument can take state name , district name or city name.If data is not found of that particular state,district or city a no data found error will be printed in console

Get the filtered data

Get data of a particular state

Here the data of states can be seen as daily or cumulative basis

filter_data=Data(init)

Maharashtra_data=filter_data.get_dataset_state(state='mh',daily=False)   #Get cumulative data of Maharashtra.state name or state code both are applicable.

Get data for a particular date

date_data=filter_data.get_dataset_by_date(date=5/06/2020)

Get cumulative data between two dates

between_date=filter_data.get_cum_dataset_between_date(4/6/2020, 9/6/2020, by='Total Confirmed')   #Gives the cumulative data between 4/6/2020 and 9/6/2020 of Total Confirmed data.

*by parameters are 'Total Confirmed' or 'Total Recovered' or 'Total Death'.

Get daily count between two dates

between_date_daily=filter_data.get_count_between_date(4/6/2020, 9/6/2020, by='Confirmed')   #Gives the daily data between 4/6/2020 and 9/6/2020 of Confirmed data.

*by parameters are 'Confirmed' or 'Recovered' or 'Death'.

Get daily data for a particular date

date_daily=filter_data.get_count_by_date(by='Confirmed',date='8/5/2020')   #Gives the daily data on 5/5/2020 of Confirmed data.

*by parameters are 'Confirmed' or 'Recovered' or 'Death'.

Get top/bottom ranked states or days

filter_data.rank(num=5, by='Total Recovered', kind='top', cumulative=True, date='8/5/2020')   #Gives top 5 states having highest recovery till date 8/5/2020

rank(num=5, by='Total Recovered', kind='top', cumulative=False, date='8/5/2020')   #Gives top 5 states having highest recovery on date 8/5/2020

*by parameters are 'Total Confirmed' or 'Total Recovered' or 'Total Death'.If cumulative is False and date is not given then it returns a dictionary.
kind parameter can take 'top' and 'bottom'

Get test Dataset

filter_data.tested_subject_data(date=None,state=None)   #Gives all tested data for all dates and for all states

filter_data.tested_subject_data(date='9/5/2020',state=None)   #Gives tested data for all states on 9/5/2020

filter_data.tested_subject_data(date='9/5/2020',state='Maharashtra')   #Gives tested data for Maharashtra on 9/5/2020 (state code also applicable)

Get graphical representation of the data

Initialize visualizer

graph=visualizer(init)

Visualize all confirmed, recovered,deceased data

graph.whole(title='corona spread of India',daily=False)   #Gives a subplot consisting of 3 graphs using cumulative data having title 'corona spread of India'

graph.whole(title='Daily corona spread of India',daily=True)   #Gives a subplot consisting of 3 graphs using Daily data having title 'Daily corona spread of India'

Visualize last and first n days confirmed, recovered,deceased data

graph.tail(num=10, title='last 10 days',daily=False)   #Gives a subplot consisting of 3 graphs using last 10 days cumulative data having title 'last 10 days'

graph.head(num=10, title='first 10 days',daily=False)   #Gives a subplot consisting of 3 graphs using first 10 days cumulative data having title 'last 10 days'

Visualize data between two dates

graph.graph_by_date(startDate=2/2/2020, endDate=5/6/2020, state='wb',title='WestBengal', daily=True)   #Gives a subplot consisting of 3 graphs using data between 2/2/2020 and 5/6/2020 of WestBengal on daily basis

Command Line Interface

Covdata package is also implemented with CLI commands that will be helps the users to get information more quickly.

#get help
covdata -h

#get dataset of user choice
covdata -f Total_Confirmed --options show_data

#save the above data as csv
covdata -f Total_Confirmed --options show_data --save \where\to\save\filename.csv

#get demographic overview by place and date
covdata -p wb -d all --options demography #please use state code.here 'wb' means 'West bengal'

#get above data for fast n rows for simplicity
covdata -p wb -d all --options demography -H 10 #head(10)

#get data by state
covdata -s mp --options state_dataset #for madhyapradesh

#get daily count data for any state
covdata -s mp --options state_dataset -D y #daily basis count for madhyapradesh.

#get daily count for all states of total confirmed/total recovered/total death
covdata -f Total_Confirmed --options state_dataset #Total Confirmed cases for daily basis for all states.

#get data by date
covdata -d 10/4/2020 --options data_by_date

#get count by date
covdata -d 11/4/2020 -f Total_Recovered --options count_by_date

#get top n values in India
covdata -r top -n 10 -f Total_Confirmed --options rank #top 10 values of total confirmed cases as cumulative

#get top 10 values of Total Recovered on the basis of daily counts for any date
covdata -r top -n 10 -f Total_Recovered -D y --options rank -d 15/3/2020

#get bottom 10 values of Total Recovered cases as cumulative counts for any date
covdata -r bottom -n 10 -f Total_Recovered -D n --options rank -d 15/3/2020

#get top 10 values of Total Recovered cases as cumulative counts for any date
covdata -r top -n 10 -f Total_Recovered -D n --options rank -d 15/3/2020

#get top 10 values of Total Recovered cases as cumulative counts for all dates if date is not mentioned
covdata -r top -n 10 -f Total_Recovered -D n --options rank

#get cumulative data between two given dates
covdata -d 15/3/2020-30/3/2020 -f Total_Recovered --options cumulative_between_date #Total recovered cumulative data between 15/3/2020 and 30/3/2020

#get daily count between two dates
covdata -d 15/3/2020-30/3/2020 -f Total_Confirmed --options count_between_date #Daily count data between 15/3/2020 and 30/3/2020

#graph of whole data
covdata -g whole -D y --options graph #plot the whole data with dailly counts

#graph whole data of any state
covdata -g whole -D y -s wb --options graph #plots the daily graph of west bengal

#graph of 1st n data of any selected state ,if state not mentioned all together state count will be shown
covdata -g head -n 15 -D y -s wb --options graph #plots 1st 15 days daily counts for west bengal
covdata -g tail -n 15 -D n -s wb --options graph #plots last 15 days cumulative counts for west bengal

#get a daily or cumulative graph between two dates for any states
covdata -g graph_by_date -d 25/3/2020-4/4/2020 -D y -s mh --options graph

#if state is not mentioned graph is shown for all states together
covdata -g graph_by_date -d 25/3/2020-4/4/2020 -D y --options graph

#latitude vs confirmed cases
covdata -g latitude --options graph

#similarly -g tail will work same and please use state code.Terminal feature demonstrates some methods only that are written in --options flag(type 'covdata -h' to see options). -H -T and --save flag can be used for every method that returns a dataframe not a graph.

Covidindia Server

User can up a local server where all the data that are collected from source are given in an interactive way.

How to start the server

covdata cov-server #to start the server with default port

covdata cov-server -b :8000 #to bind the port at 8000
